This become my final rendering and refined version of my Robot Character Design.
This design ended up with a predominantly purple and yellow colour palette for both the character and thruster.

Compared to previous iterations, this design combines both the thruster design and character design into one,
as this approach allowed for a more simplified character design.

It shows the combined character and thruster design in both an active and diabled state for the thruster.
